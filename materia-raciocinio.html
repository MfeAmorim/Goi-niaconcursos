🔢 1. Raciocínio Lógico

O raciocínio lógico é a capacidade de identificar padrões, relações e estruturas para chegar a conclusões corretas. Nos concursos, costuma aparecer em quatro grandes blocos:

  • Sequências lógicas: identificar o próximo elemento de uma série numérica ou figurativa. Exemplo: 2, 4, 6, 8, 10 (progressão aritmética de razão +2).
  • Analogias: identificar a relação entre pares de conceitos (A está para B assim como C está para ?).
  • Proposições lógicas: avaliar se uma afirmação é Verdadeira (V) ou Falsa (F) com base em conectivos como "e", "ou", "se… então", "não".
  • Diagramas lógicos: relacionar conjuntos (tipo diagrama de Euler-Venn) para tirar conclusões sobre grupos.

💡 Dica de prova: Em sequências, sempre calcule a diferença entre termos consecutivos primeiro. Se a diferença for constante → Progressão Aritmética (PA). Se a razão entre termos for constante → Progressão Geométrica (PG).

➕ 2. Operações Fundamentais

As quatro operações básicas são cobradas em contextos de problemas:

  • Adição e Subtração: base de todos os cálculos. Atenção ao "vai um" e ao "empresta".
  • Multiplicação: soma repetida. Importante dominar a tabuada até 12.
  • Divisão: distribuição igualitária. O resultado é o quociente; o que sobra é o resto.
Resultado = 15 + 27 = 42  |  2x = 10 → x = 5

📊 3. Porcentagem e Regra de Três

Porcentagem (%) significa "por cem". Para calcular x% de um número N:

Valor = (x ÷ 100) × N

Regra de três simples: usada quando existe proporcionalidade entre duas grandezas.

  • Se grandezas são diretamente proporcionais: quando uma aumenta, a outra aumenta.
  • Se grandezas são inversamente proporcionais: quando uma aumenta, a outra diminui.

Regra de três composta: envolve 3 ou mais grandezas simultaneamente.

⚖️ 4. Razões e Proporções

Razão é a comparação entre dois valores por divisão: a/b. Proporção é a igualdade de duas razões: a/b = c/d.

Produto dos meios = Produto dos extremos  →  b × c = a × d

📈 5. Estatística Básica

  • Média Aritmética: soma de todos os valores dividida pela quantidade de valores.
    Exemplo: média de 5, 7, 9 e 11 = (5+7+9+11) ÷ 4 = 32 ÷ 4 = 8
  • Mediana: valor central após ordenar os dados. Se houver número par de dados, é a média dos dois centrais.
  • Moda: valor que aparece com maior frequência no conjunto de dados.

💡 Dica de prova: Para calcular a média, some tudo e divida pela quantidade de elementos. Nunca esqueça de ordenar os dados antes de calcular a mediana!

🚗 6. Problemas de Tempo, Distância e Velocidade

A fórmula fundamental é:

Velocidade = Distância ÷ Tempo  |  V = D / T
  • Para encontrar a distância: D = V × T
  • Para encontrar o tempo: T = D ÷ V
  • Atenção às unidades: km/h vs m/s (1 m/s = 3,6 km/h)
