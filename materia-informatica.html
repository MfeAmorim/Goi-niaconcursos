🖥️ 1. Sistema Operacional Windows

O Windows é o sistema operacional mais usado em computadores pessoais e no ambiente corporativo público. Ele gerencia hardware e software, oferecendo uma interface gráfica (GUI) para o usuário.

  • Gerenciamento de arquivos: criação, cópia, movimentação, exclusão de arquivos e pastas via Windows Explorer.
  • Área de trabalho: atalhos, barra de tarefas, menu Iniciar.
  • Painel de Controle: configurações de sistema, usuários, impressoras, rede.
  • Lixeira: armazena arquivos excluídos temporariamente antes da exclusão definitiva.

💡 Dica de prova: Ctrl + C = Copiar | Ctrl + X = Recortar | Ctrl + V = Colar | Ctrl + Z = Desfazer | Ctrl + S = Salvar | Ctrl + P = Imprimir.

📝 2. Microsoft Word

Editor de texto do pacote Office, utilizado para criar e formatar documentos. Extensão: .docx

  • Formatação: fonte, tamanho, negrito, itálico, sublinhado, alinhamento, espaçamento.
  • Menu Página Inicial: contém as opções de fonte e parágrafo mais usadas.
  • Estilos: padrões de formatação (Título 1, Título 2, Normal).
  • Tabelas: inseridas via Menu Inserir → Tabela.
  • Correção ortográfica: ativada por padrão — sublinhado vermelho = erro ortográfico; azul = gramatical.
  • Mala direta: recurso para envio de documentos personalizados em massa.

📊 3. Microsoft Excel

Planilha eletrônica para organização de dados, cálculos e geração de gráficos. Extensão: .xlsx

  • Célula: interseção de uma linha (número) e coluna (letra). Ex: A1, B2.
  • Fórmulas: sempre começam com =
=SOMA(A1:A10) → Soma intervalo
=MÉDIA(A1:A10) → Média aritmética
=SE(condição;"V";"F") → Condicional
=MÁXIMO() / =MÍNIMO() → Maior / menor valor
  • Gráficos: criados selecionando dados e acessando Menu Inserir → Gráfico.
  • Tabela dinâmica: resumo interativo de grandes volumes de dados.
  • Formatação condicional: destaca células automaticamente com base em regras.

📽️ 4. Microsoft PowerPoint

Programa para criação de apresentações de slides. Extensão: .pptx

  • Slide: cada página da apresentação.
  • Layout: modelo pré-definido para cada slide (título, conteúdo, comparação).
  • Animações: efeitos aplicados a elementos dentro do slide.
  • Transições: efeitos aplicados entre slides.
  • Apresentação: iniciada com a tecla F5 (do início) ou Shift + F5 (slide atual).

🌐 5. Internet e Navegadores

  • URL (Uniform Resource Locator): endereço de uma página na web. Ex: https://www.google.com
  • HTTP / HTTPS: protocolo de transferência — HTTPS é a versão segura (criptografada).
  • Navegadores: Google Chrome, Microsoft Edge, Mozilla Firefox.
  • Modo anônimo: não salva histórico, cookies ou cache localmente — não garante anonimato total na internet.
  • Cookies: arquivos salvos pelo navegador para lembrar preferências e sessões.
  • Cache: arquivos temporários que aceleram o carregamento de páginas já visitadas.

🔒 6. Segurança da Informação

Baseia-se na tríade CID:

  • Confidencialidade: apenas pessoas autorizadas acessam a informação.
  • Integridade: a informação não é alterada sem autorização.
  • Disponibilidade: a informação está acessível quando necessário.
  • Antivírus: detecta e remove malwares, vírus, trojans e spywares.
  • Firewall: filtra o tráfego de rede, bloqueando acessos não autorizados.
  • Phishing: golpe por e-mail ou site falso que tenta roubar dados.
  • Senha forte: mistura letras maiúsculas, minúsculas, números e símbolos.
  • Backup: cópia de segurança dos dados para recuperação em caso de perda.

⚠️ Dica de prova: Antivírus protege contra vírus. Firewall protege a rede. Nenhum dos dois garante 100% de segurança — boas práticas do usuário são essenciais!

📧 7. Correio Eletrônico (E-mail)

  • Remetente: quem envia a mensagem.
  • Destinatário (Para): quem recebe.
  • CC (Com cópia): outros destinatários visíveis a todos.
  • CCO (Cópia oculta): destinatários não visíveis uns aos outros.
  • Assunto: título resumido do e-mail.
  • Anexo: arquivo enviado junto com a mensagem.
  • SMTP: protocolo para envio de e-mails.
  • POP3: baixa e-mails do servidor para o dispositivo (remove do servidor).
  • IMAP: acessa e-mails no servidor sem removê-los (sincronizado em vários dispositivos).
