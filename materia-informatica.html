🖥️ Sistema Operacional Windows Windows

O Windows é o sistema operacional mais usado em computadores pessoais e no ambiente corporativo. Gerencia hardware e software via interface gráfica (GUI).

  • Gerenciamento de arquivos: criação, cópia, movimentação e exclusão via Windows Explorer.
  • Lixeira: armazena arquivos excluídos temporariamente antes da exclusão definitiva.
  • Painel de Controle: configurações de sistema, usuários, impressoras e rede.
💡 Atalhos essenciais

Ctrl+C = Copiar · Ctrl+X = Recortar · Ctrl+V = Colar · Ctrl+Z = Desfazer · Ctrl+S = Salvar · Ctrl+P = Imprimir · Alt+F4 = Fechar

📝 Microsoft Word Word

Editor de texto do pacote Office. Extensão: .docx

  • Formatação: fonte, tamanho, negrito (Ctrl+B), itálico (Ctrl+I), sublinhado (Ctrl+U).
  • Menu Página Inicial: contém opções de fonte e parágrafo.
  • Correção ortográfica: sublinhado vermelho = erro ortográfico; azul = gramatical.
  • Mala direta: envio de documentos personalizados em massa.
  • Salvar como PDF: Arquivo → Exportar → PDF.

📊 Microsoft Excel Excel

Planilha eletrônica para cálculos e dados. Extensão: .xlsx. Fórmulas sempre iniciam com =.

=SOMA(A1:A10) → Soma intervalo
=MÉDIA(A1:A10) → Média aritmética
=SE(condição;"V";"F") → Condicional
=MÁXIMO() / =MÍNIMO() → Maior / menor valor
  • Célula: interseção de linha (número) e coluna (letra). Ex: A1, B2.
  • Formatação condicional: destaca células automaticamente por regras.
  • Tabela dinâmica: resumo interativo de grandes volumes de dados.

🌐 Internet, Navegadores e E-mail Internet

  • URL: Uniform Resource Locator — endereço de uma página. Ex: https://www.google.com
  • HTTP/HTTPS: protocolo de transferência. HTTPS = versão segura (criptografada).
  • Modo anônimo: não salva histórico, cookies ou cache localmente — mas NÃO garante anonimato total.
  • Cookies: arquivos para lembrar preferências e sessões.

Protocolos de E-mail

📤 SMTP

Simple Mail Transfer Protocol — protocolo para envio de e-mails.

📥 POP3

Baixa e-mails do servidor para o dispositivo e os remove do servidor.

🔄 IMAP

Acessa e-mails no servidor sem removê-los — sincronizado em múltiplos dispositivos.

🔒 Segurança da Informação Segurança

Baseia-se na tríade CID:

🔐 Confidencialidade

Apenas pessoas autorizadas acessam a informação.

✅ Integridade

A informação não é alterada sem autorização.

⏰ Disponibilidade

A informação está acessível quando necessário.

  • Antivírus: detecta e remove malwares, vírus, trojans e spywares.
  • Firewall: filtra o tráfego de rede, bloqueando acessos não autorizados.
  • Phishing: golpe via e-mail ou site falso para roubar dados.
  • Backup: cópia de segurança para recuperação em caso de perda.
⚠️ Dica de prova

Antivírus ≠ Firewall. Antivírus combate vírus/malware. Firewall filtra tráfego de rede. Nenhum garante 100% de segurança — boas práticas são essenciais!
